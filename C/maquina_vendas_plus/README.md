## Máquina de Vendas (C)

Projeto desenvolvido durante a graduação para praticar lógica de programação em C.

### Funcionalidades
- Seleção de produtos por código
- Validação de entradas
- Sistema de pagamento com notas e moedas
- Cálculo de troco
- Menu interativo em loop

Este projeto tem como objetivo demonstrar o aprendizado da linguagem C em nível introdutório/intermediário.
